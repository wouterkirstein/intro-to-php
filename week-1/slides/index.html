What is PHP?

PHP: HyperText Preprocessor

Recursive backronym

It's a lot like JavaScript.

They share syntactical structure.

They are both scripting languages.

Where exactly PHP fits

How web servers work

PHP is executed on the server, after the request, and before the response.

PHP versus JS

The server gets a request from a client.

PHP creates the DOM, and hands it to the server.

The server then sends the whole document to the client.

The browser renders the DOM, and loads all additional assets.

Once the page is rendered and the JS is loaded, JS can manipulate the DOM.

Operators

Do stuff with variables

Arithmetic


  <?php

    // Addition
    $var = 1 + 1; // 2

    // Subtraction
    $var = 1 - 1; // 0

    // Negation
    $var = -$a; // The oposite of $a

    // Division
    $var = 2 / 1; // 2

    // Multiplication
    $var = 2 * 3; // 6

    // Modulus (Remainder)
    $var = 20 % 6; // 2

    // Exponentiation
    $var = 10 ** 2; // 100

    // Increment & Decrement
    $a = 12;
    $a++; // 13
    $a--; // 12


  ?>

Manual: Arithmetic Operators

Comparison


  <?php

    // Less than
    $var = (1 < 2); // true
    $var = (2 <= 2); // true

    // Greater than
    $var = (1 > 2); // false
    $var = (2 >= 2); // true

    // Equality
    $var = (2 == 2); // true
    $var = (2 != 2); // false

  ?>

Manual: Comparison Operators

Logic


  <?php

    $a = true;
    $b = 123;

    // AND
    $c = $a && ($b == 1); // false
    $c = $a && ($b == 123); // true

    // OR
    $c = ($a == true) || ($b = 1); // true
    $c = ($a == false) || ($b = 123); // true

    // NOT
    $c = !($b == 1); // true

  ?>

Foreach Loop #1


  <?php

    $people = array ("Becky", "Graham", "Sean", "Duncan");

    foreach ($people as $person) {

      echo $person;

    }

    // Will print: "BeckyGrahamSeanDuncan"

  ?>

Foreach Loop #2


  <?php

    // Associative array
    $array = array("a" => 1, "b" => 2, "c" => 3);

    foreach ($array as $key => $value) {

      echo "$key: $value // ";

    }

    // Will print: "a: 1 // b: 2 // c: 3 // "

  ?>

Debugging

Knowing how to debug safely makes your life better.

Because PHP is a scripting language, it can't really be stepped through like Obj-C, Java, C#, etc.

This is one of the biggest drawbacks with PHP.

Therefore, we have to stop the code's execution manually to check that we're still on track.

Debug method

1 of 2


  <?php

    $a = array ('a' => 'apple', 'b' => 'banana', 'c' => array ('x', 'y', 'z'));

    echo "<pre>";
    print_r($a);
    echo "</pre>";
    
    die();

  ?>

What is Git?

It's a Version Control System.

Never lose work again! Save every file, and every step along the way, of your entire project.

It allows you to work with other people on the same code, without overwriting each other's work.

How does Git work?

There are three (conceptual) areas in any Git project:

  • Repository
  • Working Directory
  • Staging Area
